This is a library and Jupyter Notebook for detecting and measuring wheat grains in transmitted light.


The yolo26m model trained for detecting wheat grains and empty cells on a perforated plate viewed in transmitted light can be downloaded at:
https://huggingface.co/MikhailBazhenov/yolo_for_wheat_grains/tree/main
Add the model file to the same folder with other files of this repository.
